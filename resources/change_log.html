Version: 1.0.3

  • Updated: Forge 9.10.0.804.
  • New: Config option to enable hunger when difficulty is set to peaceful. Default value false to preserve vanilla behavior.
  • New: Four new config options that control the how far health will fall when the player is starving. There is one option for each difficulty level. Note that the peaceful difficulty option only applies if hunger is enabled for peaceful difficulty.
  • New: Added the /dif command. Works the same as the vanilla /difficulty command, but will display the current difficulty when called with no arguments.
  • New: Added /hunger command. Takes no arguments and displays the values loaded from the mod's config file.
  • Code: Rewrote the mod to replace the entire FoodStats class, instead of just making the exhaustion variable public.
